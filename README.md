🎓 Pooja Kumari Academic Portfolio Dashboard

An interactive academic result dashboard built using HTML, CSS, and JavaScript to showcase semester-wise performance, SGPA progression, subject-wise grades, and official result PDFs in a modern UI.

✨ Features

- 📊 Final CGPA overview
- 🎞️ Semester-wise filmstrip timeline
- 📈 SGPA progression chart
- 🔍 Subject search across all 8 semesters
- 📄 Embedded PDF result viewer
- 📱 Fully responsive design
- 🎨 Modern dark UI with animation effects

---

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts

📌 Dashboard Sections

- Rendering
- Python Programming
- 3D Modelling

3. Semester Timeline

Semester-wise SGPA, CGPA, and subject grades shown in filmstrip style.

4. SGPA Progress Chart

Visual representation of semester performance.

5. Official Result PDF

Integrated PDF viewer with download support.
